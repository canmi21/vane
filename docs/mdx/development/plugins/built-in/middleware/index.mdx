title: Definition
description: Plugins that inspect, modify, or branch the flow.

---

Middleware plugins are the workhorses of the Vane engine. They implement the `GenericMiddleware` or `HttpMiddleware` traits.

They generally fall into two categories:

1.  **Logic Units**: Pure control flow (e.g., Matcher, Rate Limit).
2.  **Drivers**: Plugins that perform I/O (e.g., Fetch Upstream, CGI) but technically run as middleware to allow for post-processing or chaining.
