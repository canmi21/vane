This document outlines the planned features and architectural improvements for Vane.

## Core Features

- [ ] **ACME Support**: Built-in Let's Encrypt integration for automatic certificate management.
- [ ] **DTLS Support**: L4+ inspection for Datagram Transport Layer Security (SNI routing for UDP).
- [ ] **L7 DNS**: Full DNS protocol parsing and manipulation layer (similar to HTTP).
- [ ] **Management API**: Comprehensive REST API for runtime configuration and status monitoring.

## Ecosystem

- [ ] **Web Dashboard**: A React-based admin panel consuming the Management API.
- [ ] **Prometheus Exporter**: Native metrics endpoint for monitoring.
- [ ] **WASM Plugins**: Support for WebAssembly-based middleware for safe, high-performance extensions.
