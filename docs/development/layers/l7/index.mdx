Layer 7 (L7) is where Vane performs full **HTTP Termination**. Unlike L4/L4+, it parses the entire request (Headers & Body), allowing for deep inspection, transformation, and synthetic responses.

## The Unified Adapter

Vane's L7 Architecture is designed around **Protocol Unification**. The Flow Engine does not know (or care) whether a request came from HTTP/1.1, HTTP/2 multiplexed stream, HTTP/3 QUIC stream, or a virtual internal channel.

**Source**: `src/layers/l7/http/`

### Flow Execution

The Flow Engine executes middleware. Middleware can inspect headers or force-buffer the body on demand.

<Mermaid
  chart="
graph LR
    Container --> Flow[Flow Executor]
    Flow --> Mid1[Middleware 1]

/>

</Step>

### Termination

A Terminator signals completion. It sends headers via a oneshot channel and streams the body back to the protocol engine.

<Mermaid
  chart="
graph LR
    Terminator -->|Signal| Channel[Oneshot TX]

    Channel --> Adapter[Wait RX]
    Body --> Adapter
    Adapter -->|Serialize| Client
"
/>

</Step>
