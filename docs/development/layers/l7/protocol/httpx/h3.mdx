## Implementation

**Source**: `src/layers/l7/http/h3.rs`

HTTP/3 runs over QUIC and requires a dedicated driver loop due to the complexity of ownership management in the `h3` crate.

<Mermaid
  chart="
graph LR
    Quic[QUIC Stream] <--> Driver[Driver Loop]
    Driver -->|Push Chunk| Chan[MPSC Channel]

    Adapter -->|Body| Container[Container]
    Container -->|Response| Driver

<Steps>
<Step>
### Driver Loop
A `select!` loop runs for each request stream.
- **Pump Request**: Reads from QUIC, sends to `mpsc` channel.
- **Wait Response**: Waits for `Container` to signal response headers.
- **Pump Response**: Reads from `VaneBody`, writes to QUIC.
</Step>

<Step>
### Channel Adapter
`H3BodyAdapter` bridges the gap between the `mpsc` channel and the `http_body::Body` trait.

**Goal**: Allows the Container to treat H3 streams exactly like H1/H2 streams.

</Step>
</Steps>
