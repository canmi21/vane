## Implementation

**Source**: `src/layers/l7/http/httpx.rs`

HTTP/2 support is integrated into the same `httpx` adapter as HTTP/1.1, utilizing Hyper's automatic protocol detection.

	chart="

<Steps>
<Step>
### Server Side (Ingress)
- **Multiplexing**: Handled transparently by Hyper.
- **Concurrency**: Each stream spawns a new Tokio task executing the L7 Flow.
- **Prioritization**: Basic stream prioritization supported by `h2` crate.
</Step>

<Step>
### Client Side (Upstream)
- **Feature**: `h2upstream` (enabled by default).
- **Pooling**: Uses a sophisticated connection pool (`src/plugins/l7/upstream/pool.rs`).
- **Reuse**: Reuses the same TCP connection for multiple concurrent requests.
</Step>
</Steps>
