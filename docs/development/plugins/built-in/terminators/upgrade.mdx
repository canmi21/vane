- **ID**: `internal.transport.upgrade`
- **Source**: `src/plugins/protocol/upgrader/upgrade.rs`
- **Trait**: `Terminator`
- **Layer**: **L4**, **L4+** (Transitions to Higher Layer)

### Logic

Signals the engine to "upgrade" the current connection (e.g., from raw TCP to TLS, or UDP to QUIC). This stops the current L4 flow and starts a new flow in the higher layer.

### Parameters

| Name       | Required | Description                                      |
| :--------- | :------- | :----------------------------------------------- |
| `protocol` | Yes      | Target protocol example (`tls`, `http`, `quic`). |
| `cert`     | No       | Optional SNI override for TLS termination.       |
