# Vane Banner

Shared `--version` banner printer for the [vane](https://vane.canmi.app) CLI
and daemon. Keeps the colour palette + layout in one place so `vane -v` and
`vaned -v` stay byte-for-byte identical.

Part of the [vane](https://github.com/canmi21/vane) workspace. See the main
repository for documentation and architecture specs.
