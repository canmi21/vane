# Vane Daemon

The [vane](https://vane.canmi.app) proxy daemon: L4 forwarding, HTTP any-bridge
(HTTP/1.1 / HTTP/2 / HTTP/3 in any combination), WebSocket upgrade, TLS, and
WASM plugins.

Part of the [vane](https://github.com/canmi21/vane) workspace. See the main
repository for documentation and architecture specs.
