# Vane Management

Management protocol for the [vane](https://vane.canmi.app) proxy: wire format,
verb schemas, server, and client. One protocol over two transports — Unix
socket (default) and HTTP-over-TCP (opt-in remote).

Part of the [vane](https://github.com/canmi21/vane) workspace. See the main
repository for documentation and architecture specs.
